## 🚀 **즉시 배포 가능**

### 1️⃣ **Google Sheets 설정** (5분)
```bash
1. Google Sheets 새 시트 생성
2. google-apps-script/main.gs 코드 복사
3. Apps Script 웹앱 배포
```

### 2️⃣ **환경 설정** (2분)
```bash
# .env.local 파일 생성
VITE_GOOGLE_SHEETS_API_URL=your_script_url
```

### 3️⃣ **배포** (3분)
```bash
# Vercel 배포
npm run build
vercel --prod
```

**⏱️ 총 소요시간: 10분**
